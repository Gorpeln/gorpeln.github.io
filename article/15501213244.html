Sublime Text 3安装使用 MAC

时间：2019-02-14

    个人分类： Software MAC     标签： Sublime Text3
转载请注明出处：https://gorpeln.com/article/15501213244

Sublime Text：一款具有代码高亮、语法提示、自动完成且反应快速的编辑器软件，不仅具有华丽的界面，还支持插件扩展机制，用她来写代码，绝对是一种享受。相比于难于上手的Vim，浮肿沉重的Eclipse，VS，即便体积轻巧迅速启动的Editplus、Notepad++，在SublimeText面前大略显失色，无疑这款性感无比的编辑器是Coding和Writing最佳的选择，没有之一。

安装

官网：http://www.sublimetext.com/3 选择对应的版本，点击下载安装

插件

1、安装 package control

功能：安装插件的工具，有了它，以后安装其他插件更方便

第一步：按 Ctrl + ~ 键，调出 sublime 的控制台；

第二步：在调出的控制台输入如下指令（需要科学上网）

import urllib.request,os;pf='Package Control.sublime-package';ipp=sublime.installed_packages_path();urllib.request.install_opener(urllib.request.build_opener(urllib.request.ProxyHandler()));open(os.path.join(ipp,pf),'wb').write(urllib.request.urlopen('http://sublime.wbond.net/'+pf.replace(' ','%20')).read())

回车，稍等片刻，如果在 preferences 里有 package control 就说明安装成功了；

第三步：重启Sublime Text 3，查看Perferences->package settings中是否有package control这一项，如果有，则安装成功

安装插件：Ctrl+Shift+P调出命令面板，在输入框输入Install Package ，然后在新弹出的窗口中查询要安装的插件，点击需要安装的插件，即可；

并可在Perferences->package settings 中看到对应的包名。

删除插件：Ctrl+Shift+P调出命令面板，输入remove，调出Remove Package选项并回车，选择要删除的插件即可。

更新插件：输入：upgrade packages，后回车，如果有列表出来则有插件需要更新，然后点击需要升级的插件即可；

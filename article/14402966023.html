Markdown

Markdown 是一种方便记忆、书写的纯文本标记语言，用户可以使用这些标记符号以最小的输入代价生成极富表现力的文档。

1. 标题

使用#显示标题

# 这是一个一级标题
## 这是一个二级标题
### 这是一个三级标题

你也可以选择在行首加#号表示不同级别的标题 (H1-H6)，例如：# H1, ## H2, ### H3，#### H4。

2. 斜体、粗体、删除线

语法 效果
*斜体1* 斜体1
_斜体2_ 斜体2
**粗体1** 粗体1
__粗体2__ 粗体2
这是一个 ~~删除线~~ 这是一个 删除线
***斜粗体1*** 斜粗体1
___斜粗体2___ 斜粗体2
***~~斜粗体删除线1~~*** 斜粗体删除线1
~~***斜粗体删除线2***~~ 斜粗体删除线2

斜体、粗体、删除线可混合使用

3. 链接

外链接

使用 [描述](链接地址) 为文字增加外链接。

# 语法 效果
1 [CSDN](https://blog.csdn.net/chen_gp_x) CSDN
2 [GitHub](https://github.com/Gorpeln) GitHub

图片链接

因为图片本身和链接本身都支持URL标识符的形式，所以图片链接也可以很简洁:

 ![alt](URL title)

alt和title即对应HTML中的alt和title属性（都可省略）：

  • alt表示图片显示失败时的替换文本
  • title表示鼠标悬停在图片时的显示文本（注意这里要加引号）
  • 后面添加[target]可以完成跳转
# 语法 效果
1 ![](https://gorpeln.com/assets/img/social/meLogo.png)
2 ![](https://gorpeln.com/assets/img/social/csdnLogo.png "csdn")[csdn]
3 ![github-logo][github]

锚点

其实呢，每一个标题都是一个锚点，和HTML的锚点（#）类似，比如我们

语法 效果
[回到顶部](#markdown) 回到顶部
[去到尾部](#end) 去到尾部

不过要注意，标题中的英文字母都被转化为小写字母了。

4. 代码块、高亮

  • 在连续几行的文本开头加入1个Tab或者4个空格。
  • 使用一对各三个的反引号

代码块

#import "ViewController.h"

@interface ViewController ()
@end

@implementation ViewController

- (void)viewDidLoad {
    [super viewDidLoad];

    NSLog(@"Hello World");
    
}
@end

文字高亮

文字高亮功能能使行内部分文字高亮，使用一对反引号。 语法：

`linux` `iOS` `HTML5` `小程序` `php` `APIClound`

效果：linux iOS HTML5 小程序 php APIClound

5. 列表

无序列表

使用 *，+，- 表示无序列表。 示例：

- 无序列表项 
- 无序列表项 
- 无序列表项 
    - 无序列表
    - 无序列表

效果：

  • 无序列表项
  • 无序列表项
  • 无序列表项
    • 无序列表
    • 无序列表

有序列表

就是在数字后面加一个点，再加一个空格。不过看起来起来可能不够明显。 示例：

1. 有序列表项 
2. 有序列表项 
3. 有序列表项

效果：

  1. 有序列表项
  2. 有序列表项
  3. 有序列表项

复选框列表

你可以使用这个功能来标注某个项目各项任务的完成情况。

7. diff语法

版本控制的系统中都少不了diff的功能，即展示一个文件内容的增加与删除。 GFM中可以显示的展示diff效果。使用绿色表示新增，红色表示删除。

其语法与代码高亮类似，只是在三个反引号后面写diff， 并且其内容中，以 +开头表示新增，-开头表示删除。

效果如下：

+ 运筹帷幄之中，决胜千里之外
- 运筹帷幄之中，决胜千里之外
